Methods

1

Random Undersampling

Random undersampling is a process we used to bring down the number of non-fraudulent data to match that of fraudulent data. This is necessary to obtain a more accurate and reliable result as we had a completely imbalanced dataset to process.

2

K-fold Cross Validation

K-fold cross validation is a type of model evaluation method, where k is an integer that you choose, and the data is divided into k amount of subsets, and is iterated k amount of times. Each time, one of the subsets will be used as a test set and the rest as training sets. The computer will then take the mean score of all the iterations.

3

Logistic Regression

For the algorithm, we originally utilised logistic regression, which was perfect for what we set out to achieve, as our project aimed to produce a discrete outcome, which was either fraudulent or non-fraudulent. In addition to that, logistic regression is probability based, able to analyse the probability of a case being either fraudulent or non-fraudulent.

4

Support Vector Machine

After the semi-finals, we decided to switch from logistic regression to the Support Vector Machine. This is where each data item is plotted as a point in “n”-dimensional space, where n is the amount of features it has. Classification of the cases is performed by finding the optimum hyper-plane that can differentiate the 2 classes effectively, as seen from the diagram attached here.
