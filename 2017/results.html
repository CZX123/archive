Test 1

1
Tested out the machine with the entire skewed dataset, effectively using it as control.
2
Logistic regression used as algorithm for Machine Learning.
3
K-folding was used, due to large amounts of data.

Result 1

From the confusion matrix, it is evident that while the machine was able to correctly predict a most of the non-fraudulent data, it is clearly unable to accurately predict fraudulent data, as it predicted 56 out of the 147 fraudulent cases wrongly. This is due to the fact that the machine has little exposure to the fraudulent data.

Test 2

1
The dataset is undersampled with random undersampling.
2
Logistic regression used as algorithm for Machine Learning.
3
K-folding was used, due to large amounts of data.
4
Once the machine is done learning, it was tested with the whole skewed dataset.

Result 2

After bringing down the number of non-fraudulent data to an equal number of 492, the machine is able to correctly predict 134 non-fraudulent cases out of 149 total. While the first result showed a total inability to predict fraudulent cases, this result suggested the opposite, with the machine being able to correctly predict 138 fraudulent cases out of 147.

After the machine was done learning, we used it on the entire dataset, which also yielded favourable results, with the machine correctly predicting 135 fraudulent cases out of a total of 147. Hence, we have decided that undersampling is a very effective method.

Test 3

1
The dataset is undersampled with random undersampling.
2
Support Vector Machine used as algorithm for Machine Learning.
3
K-folding was used, due to large amounts of data.
4
Once the machine is done learning, it was tested with the whole skewed dataset.

Result 3

With the use of support vector machine, as well as undersampling, the results were slightly better than that of test 2, but mostly around the same.

However, when the code is tested on the heavily skewed dataset, the results were much more accurate than that of test 2, being able to accurately predict more of the non-fraudulent cases.

Conclusion

After conducting three different tests, we have come to realise that for our case, the most effective program is when we implemented the methods of random undersampling, K-fold cross validation and the support vector machine instead of logistic regression.
